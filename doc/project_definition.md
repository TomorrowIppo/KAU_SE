# 프로젝트 정의서: 재난 대비 P2P 드론 메쉬 네트워크 (Resilience Drone-P2P)

## 1. 프로젝트 개요
지진, 홍수 등 재난 상황에서 기존 기지국이 파괴되었을 때, 여러 대의 드론을 띄워 P2P 방식의 임시 통신망을 구축하고 구조 대원 및 고립자 간의 데이터 통신을 지원하는 가상 네트워크 시스템 구축을 목표로 한다.

## 2. 추진 배경 및 필요성
* **통신 인프라 마비**: 재난 발생 시 중앙 집중형 기지국 파괴로 인한 골든타임 확보 실패.
* **지형적 제약**: 산악 지형이나 침수 지역 등 구조대 진입이 어려운 곳에서의 통신 필요성.
* **중앙 서버 부재**: 고정된 서버 없이 드론 개체(Node)들이 스스로 망을 구성하는 탈중앙화 기술 절실.

## 3. 주요 기능 범위 (Scope)
* **P2P 메쉬 라우팅**: 드론 간 동적 경로 탐색 및 데이터 릴레이 전송.
* **군집 비행 제어**: 드론 간 간격을 유지하며 통신 범위를 최적화하는 위치 제어 시뮬레이션.
* **긴급 메시지 전송**: 텍스트, 위치 정보(GPS), 저대역폭 이미지 전송.
* **에너지 인식 노드 관리**: 배터리 잔량이 부족한 드론을 감지하고 대체 노드로 교체 요청.
* **보안 통신**: 비인가 드론의 망 접속 차단 및 전송 데이터 암호화.

## 4. 기대 효과
* 기존 인프라 없이도 즉각적인 광대역 통신 커버리지 확보.
* 드론 한두 대가 추락해도 전체 망이 유지되는 높은 생존성 보장.
* 신속한 정보 공유를 통한 재난 구조 효율 극대화.
