# 프로젝트 관리 계획서

## 1. 서론
### 1.1 프로젝트 목표
극한 재난 환경에서 드론을 활용한 고신뢰성 P2P 메쉬 통신 프로토콜 개발 및 검증.

## 2. 개발 계획
### 2.1 주요 개발 내용
- P2P 동적 라우팅 및 릴레이 알고리즘 설계.
- 에너지 효율적 노드 관리 및 우선순위 큐(Priority Queue) 구현.
- TelloEdu 및 가상 노드 시뮬레이션 환경 구축.

### 2.2 WBS (일정 요약)
- 1-3주: 요구사항 상세화 및 프로토콜 설계.
- 4-8주: 메쉬 라우팅 및 노드 관리 모듈 개발.
- 9-11주: 시뮬레이션 및 필드 테스트 (지연시간 500ms 검증).
- 12주: 최종 결과 보고서 산출 및 프로젝트 완료.

## 3. 품질 및 관리 계획
- 품질 보증: 노드 이탈 테스트(5초 이내 복구) 상시 수행.
- 산출물 관리: Github 저장소 및 마크다운 기반 문서 관리.

## 4. 개발 환경
- HW: TelloEdu 드론, Ubuntu 노트북.
- SW: Linux (Ubuntu 20.04), ROS Noetic, Python.
